Presentations

Building a Personal Website with Bootstrap

In this hands on workshop, we will design a personal website using HTML, CSS, and JavaScript. We will learn how to incorporate Bootstrap (http://getbootstrap.com/) design templates for features like typography, forms, buttons, navigation and more. Lastly, we will host our web app in the cloud using Azure (https://azure.microsoft.com) so we can share it with the world.

Pokémon Go Make a Website

Learn the basics of HTML and CSS to create a Pokémon themed website. I’ll cover how to add text, images and style (fonts/colors) to your locally hosted website in this hands-on lab. We will use the editor Visual Studio Code for students with Macs or Windows (download @ www.code.visualstudio.com) or jsFiddle.net for students with Chromebooks. Computer needed. Artistic freedom granted for students not interested in Pokémon. Best for students 12+.
